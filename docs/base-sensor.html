BaseSensor ​

BaseSensor is an extendable base class to ease the process of creating custom sensors. It does not do anything by itself, but it does implement the Sensor API and provides you some protected helper methods for controlling the state of the drag process. It's used by KeyboardSensor so you can check out implementation tips there.

Protected Methods ​

These methods are not meant to be exposed publicly, but they are available for any class that extends this class. You can use them to control the drag process.

ts
// Type
